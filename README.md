# Customer Churn Prediction

## 📌 Project Overview
This project predicts customer churn using machine learning models.

## 🛠 Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## 📊 Steps Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training (KNN, Decision Tree)
- Model Evaluation

## 📈 Results
- KNN Accuracy: ~91%
- Decision Tree Accuracy: ~96%
- Overfitting reduced using model tuning

## 💡 Key Insights
- Payment delay and support calls influence churn
- Long-term customers are more stable
- Decision Tree performed best

## 🎯 Conclusion
The model helps identify customers at risk of leaving and supports business decision-making.
